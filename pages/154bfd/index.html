    # Bean Mapping

    Tools that handle the build cycle and dependencies of an application.

    • Apache Maven 在新窗口打开 (opens new window) - Declarative build and dependency management that favors convention over configuration. It might be preferable to Apache Ant, which uses a rather procedural approach and can be difficult to maintain.
